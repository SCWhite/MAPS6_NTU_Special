## MAPSV6 感測項目:

| 感測項目 | 元件型號 | 單位 |
| :--- | :--- | :--- |
| PM2.5 | PMS3003 | µg/m³ |
| 溫度 | STH31 | C |
| 濕度 | STH31 | %H |
| CO2 | sense AIR S8 / honeywell CO2 M1 | ppm |
| TVOC | SGP30 | ppb |
| 光度 | TCS34725 | lux |
